question: Create a deployment called vulpix using image gcr.io/google-samples/hello-app:1.0 with three replicas. Confirm successful application deployment by outputting the result to file 
author: 
    - hemandee

tags: 
    - Core Concepts
    - Deployment
answer: |
    ```bash
    # Generate yaml from create statement
    kubectl create deployment vulpix --image=gcr.io/google-samples/hello-app:1.0 --dry-run -o yaml >> vulpix.yaml
    ```   
    ```yaml
    apiVersion: apps/v1
    kind: Deployment
    metadata:
      creationTimestamp: null
      labels:
        app: vulpix
      name: vulpix
    spec:
      replicas: 3
      selector:
        matchLabels:
          app: vulpix
      strategy: {}
      template:
        metadata:
          creationTimestamp: null
          labels:
            app: vulpix
        spec:
          containers:
          - image: gcr.io/google-samples/hello-app:1.0
            name: hello-app
            ports:
               - containerPort: 8080
            resources: {}
    status: {}
    ```
    ```bash
    # Get IPs Addresses of Pods
    $kubectl get pod -o wide
    # Confirm app endpoint
    $curl <pod ip address>:8080 > vulpix-output.txt
    
    $cat vulpix-output.txt
    Hello, world!
    Version: 1.0.0
    Hostname: vulpix-XXXXXXXX
    ```
